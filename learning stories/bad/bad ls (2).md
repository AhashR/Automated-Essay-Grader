learning story: The problem my team is building an app. But every week, our code breaks. We waste hours fixing bugs and uploading files by hand.

How it works I explain the two steps:
CI (testing: The robot tests our new code automatically. It stops bad code.
CD (building and uplaoding): If the code is good, the robot uploads it to the live website automatically.
