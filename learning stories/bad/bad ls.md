Leerverhaal: "Ik wil leren hoe ik moet coderen"
Als student wil ik leren programmeren zodat ik mijn project kan afmaken.

Context
Ons team heeft een user story die zegt: "Als gebruiker wil ik dat de app werkt." Om deze app te bouwen moet ik weten hoe ik moet programmeren. Ik heb nog nooit geprogrammeerd, maar ik heb een keer een YouTube-video gekeken en ik begreep het grootste deel. De video was 12 minuten lang. Ik denk dat dat genoeg context is.

Leerdoelen
Ik wil leren coderen.
Ik wil computers beter begrijpen.
Ik wil weten hoe Git werkt (mijn teamgenoot blijft het noemen en ik wil het niet nog een keer vragen).
Ik wil alles leren.

Hoe leer ik dat?
Ik ga het Googelen. Als dat niet werkt vraag ik het aan ChatGPT. Als ChatGPT het ook niet weet vraag ik het aan mijn teamgenoot Sem, ook al weet Sem het eigenlijk ook niet. Ik ben van plan te beginnen met leren op zondagavond om 22:30, de avond voor de sprintreview.
Ik zal de HvA Knowledgebase gebruiken, maar alleen als de eerste drie Google-resultaten niet helpen. Dat gaan ze wel doen, dus waarschijnlijk niet.
Mijn noodplan is een tutorial van YouTube kopiëren en de variabelenamen veranderen.

Reflectie
Ik heb veel geleerd deze sprint. Ik weet nu wat een for loop is. Ik heb hem niet gebruikt in mijn code, maar ik weet wat het is. Dit toont persoonlijk leiderschap.
Volgende sprint focus ik op mijn communicatievaardigheden door één Slack-berichtje per week te sturen naar mijn projectgroep.
Geschatte leertijd: 20 minuten.
Werkelijke leertijd: 4 uur in paniek om 01:00 's nachts.
